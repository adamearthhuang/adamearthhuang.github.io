# 常用指标

# PV（Page View）

页面访问量，或者说是曝光量。

# UV（Unique Visitor）

独立访客，或者说是独立用户。

# CTR（Click Through Rate）

点击通过率。

CTR = 点击量 / 曝光量

# CVR（Conversion Rate）

转化率。

CVR = 新增量 / 点击量

# CPA（Cost Per Action）

按行为付费。

CPA = 花费 / 新增量

# CPC（Cost Per Click）

按点击付费。

CPC = 花费 / 点击量

# ROI（Return On Investment）

投资回报率。

ROI = 收入 / 花费

# LTV（Life Time Value）

生命周期总价值。

LTV = 收入 / 新增量
